Torustextur
Laura Pichlmeier - 70454892

Texturieren Sie einen Torus mit einer Bildtextur.
Dazu müssen zu den Geometriedaten des Torus zunächst Texturkoordinaten hinzugefügt werden.
Interaktion: Bewegung der Kamera, wie bisher, über die Pfeiltasten.
Erweiterungen: Entwerfen und realisieren Sie eine prozedurale Textur für den Torus.

Aufgabe 9 + Erweiterung

Tipp: Nutze Shift + Pfeiltaste, um Scrollen der Seite zu verhindern.

Controls

Pfeiltasten = Steuerung der Kamera
R = Reset Canvas
The donut skin has been created by myself using Photoshop.
The Perlin Noise data used for projecting the Perlin Noise skin onto the torus has been found on github.
